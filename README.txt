README
Ultimate Tic Tac Toe game by Rahul Madanahalli
Created with Eclipse JUNO


UltimateTTT.java is the main class

TO load your own MySQL server:
1. Download XAMPP and start the MYSQL server
2. Open shell and type in mysql
3. Create a database
4. Enter this on XAMPP command line: mysqldump database_name < test.sql
	4a. ddatabase_name is name of database that you created
	4b. test.sql is a file in this repo
5. You must add the mysql-connector-java-5.1.25-bin to the build path of the project in Eclipse.
